Ошибки в коде:

Ошибка №1:
def capitalize(self, string: str) -> str:
        return string.capitalize()

Правильный вариант:
def capitalize(self, string: str) -> str:
    if string is None:
        raise AttributeError("Input string cannot be None")
    return string.capitalize()

Пояснения к ошибке:
- Нет проверки на None


Ошбика №2:
def delete_symbol(self, string: str, symbol: str) -> str:
        if self.contains(string, symbol):
            string = string.replace(symbol, "")
        return string
    
Правильный вариант:
def delete_symbol(self, string: str, symbol: str) -> str:
    if string is None or symbol is None:
        raise AttributeError("Input string and symbol cannot be None")
    return string.replace(symbol, "")

Поянения к ошибке:
- Нет проверки на None
